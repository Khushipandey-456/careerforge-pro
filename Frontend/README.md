CareerForge Pro

CareerForge Pro is a web application built to help users manage and explore career-related opportunities in a structured way. The project focuses on providing a clean UI and scalable backend integration.

Tech Stack

Frontend:

* React (with TypeScript)
* Vite
* CSS

Backend:

* Spring Boot (REST API)

Project Structure

The frontend is organized into reusable components and follows a modular structure:

* components → UI components
* pages → Page-level views
* services → API calls
* store → State management
* utils → Helper functions

Getting Started

Clone the repository and install dependencies:

npm install

Start the development server:

npm run dev

The app will run at:
http://localhost:5173/

Backend Setup

Make sure the backend service is running separately (Spring Boot application).

Update API base URLs inside the services folder if required.

Features

* Modular component-based architecture
* API integration with backend
* Clean and maintainable code structure

Notes

* Environment variables can be configured using `.env` file
* `node_modules` is ignored using `.gitignore`
